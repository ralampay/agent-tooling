# Bibliography

## Books

1. Artificial Intelligence: A Modern Approach (4th ed.) - Stuart Russell, Peter Norvig (2020). Pearson. Link: `https://www.pearson.com/en-us/subject-catalog/p/Russell-Artificial-Intelligence-A-Modern-Approach-4th-Edition/P200000003500/9780137505135`. Notes: core agent concepts, planning, and uncertainty.
2. Human-in-the-Loop Machine Learning - Robert (Munro) Monarch (2021). Manning. Link: `https://www.manning.com/books/human-in-the-loop-machine-learning`. Notes: human oversight and safe workflow design.
3. Designing Data-Intensive Applications - Martin Kleppmann (2017). O'Reilly Media. Link: `https://www.oreilly.com/library/view/designing-data-intensive-applications/9781491903063/`. Notes: reliability, state, and failure handling.
4. Release It! Second Edition: Design and Deploy Production-Ready Software - Michael Nygard (2018). Pragmatic Bookshelf. Link: `https://pragprog.com/titles/mnee2/release-it-second-edition/`. Notes: failure-first thinking and resilience patterns.

## Articles, Papers, and Docs

1. ReAct: Synergizing Reasoning and Acting in Language Models - Shunyu Yao et al. (2023). arXiv. Link: `https://arxiv.org/pdf/2210.03629`. Notes: interleaving reasoning with tool actions.
2. Toolformer: Language Models Can Teach Themselves to Use Tools - Timo Schick et al. (2023). arXiv. Link: `https://ar5iv.org/abs/2302.04761v1`. Notes: tool selection and API usage.
3. MRKL Systems: A modular, neuro-symbolic architecture that combines large language models, external knowledge sources and discrete reasoning - Ehud Karpas et al. (2022). arXiv. Link: `https://ar5iv.org/abs/2205.00445v1`. Notes: routing to specialized tools and modules.
4. Tree of Thoughts: Deliberate Problem Solving with Large Language Models - Shunyu Yao et al. (2023). arXiv. Link: `https://arxiv.org/abs/2305.10601`. Notes: explicit planning and search over steps.
5. Function Calling in the OpenAI API - OpenAI Help Center. Link: `https://help.openai.com/en/articles/8555517-function-calling-in-the-openai-api`. Notes: tool contracts and structured arguments.
